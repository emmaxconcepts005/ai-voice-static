# AI Voice Static Website

This is a simple static personal website for an AI Voice Agent & Automation Specialist.

## Why this version is easier
- No build step
- No npm install
- No framework setup
- Easy for beginners to deploy on Vercel

## Files included
- `index.html`
- `styles.css`
- `script.js`

## How to deploy on Vercel
1. Extract this zip file.
2. Create a new GitHub repository.
3. Upload the files inside this folder to the repo root.
4. Import that repo into Vercel.
5. Click Deploy.

## Important
Replace the placeholder contact info in `index.html` before publishing.
